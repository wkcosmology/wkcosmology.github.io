Taking notes using Pages

Posted on 2020-07-10 | In Blog , Tools
Pages is a powerful editing tool on Mac. In this post, I will introduce my config of the Pages for note taking.

Motivation

As a PhD student, I need to take notes of the projects, i.e. literature reviews, motivations, results etc. The basic requirements are:

  1. support figure insertion and caption
  2. support link insertion
  3. support LaTex equation

There are already many note taking software that supports the above criteria, like Evernote, OneNote etc. However, I already have DevonThink3 to organize the file, so I need a suitable file format for individual note. Markdown is a good choice(light, portable, support links and LaTex etc.). However, you need to put the figures out of the document and the figure will stay after deleting it in the markdown document. The same problem occurs to LaTex file. RTF support many functions, but it does not have a good editor. At last, I turn into Pages.

Config to Pages

Font

First I need to download the Computer Modern Serif(CMU Serif) font(default font for LaTex), which I think the best choice for documents.

Text format

In the template, I defined several text format.

  1. Heading:
    1. Style: CMU Serif, 18, Bold
    2. Style: Bullets & Lists: Numbers, Tiered Numbers, Continue from previous
    3. Layout: Borders & Rules: lines above and below, offset 2pt, width 0.5pt, black
    4. Add page break before each Heading
  2. Heading 2:
    1. Style: CMU Serif, 16, Bold
    2. Style: Bullets & Lists: Lines=1, Numbers, Tiered Numbers, Continue from previous
    3. Style: Spacing: Lines=1, Before Paragraph=8pt, After Paragraph=3pt
    4. Layout: Borders & Rules: None
    5. Add tab before each Heading 2
  3. Heading 3:
    1. Style: CMU Serif, 12, Bold, underline
    2. Style: Bullets & Lists: None
    3. Style: Spacing: Lines=1, Before Paragraph=3pt, After Paragraph=3pt
    4. Layout: Borders & Rules: None
  4. Body:
    1. Style: CMU Serif, 12
    2. Justify text
    3. Style: Spacing: Lines=1, Before Paragraph=0pt, After Paragraph=0pt
    4. Layout: Borders & Rules: None

It looks like

Figure

There are two ways to insert figures. First is just inserting figure, which also support caption after version 10.1 and you can also use LaTex in it. The second way is insert image gallery, which can contain several figures in one place and you can switch them. The image gallery also support caption for each individual figure but no LaTex in the caption. In both way, you can just drag the figure files into Pages, even they are PDF file.

Equation

Pages after version 6.1 supports LaTex equation natively.

Links

Pages support three kinds of links, as website, bookmarks and email. I use website to link to the website or DevonThink3 or Zotero links(It cannot open local files though URL).

Table of content

Pages also support TOC for convenient link to each headings(You can config the specific text format showed in TOC).

Last Word

I choose Pages since I already have DevonThink3 as my organization system. And it is simple, fast and easy to use. If you want a note management system, I will recommend Evernote or OneNote.
